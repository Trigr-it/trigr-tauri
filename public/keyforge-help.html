Welcome to Trigr

Trigr is a visual hotkey, macro and text expansion manager for Windows. No coding required — click a key, assign an action, done.

⌨

Hotkeys & Macros

Assign Ctrl+E or Ctrl+Alt+Q to type text, open apps, run sequences and more.

✦

Text Expansions

Type a trigger word, press Space, and it instantly expands into full formatted text.

⬡

App Profiles

Profiles auto-switch when you change windows — different macros per application.

◈

Macro Sequences

Chain actions together — type, press keys, wait for clicks, open URLs.

Quick start

  1. Click the Trigr icon in the system tray (bottom right of taskbar) to open the app

  2. Click Ctrl in the modifier bar above the keyboard

  3. Click any key — e.g. E — to select the Ctrl+E combination

  4. Choose an action type in the right panel and fill in the details

  5. Click Assign to Key — the key gains an amber dot

  6. Press Ctrl+E in any application — your macro fires!

System tray

Closing the Trigr window does not stop it — it keeps running in the background. Click the tray icon to reopen. The status bar shows Macros Active with a green dot when running.

Title bar — profiles

Profile tabs across the top. Active profile highlighted in amber. Linked-app profiles show a coloured dot. Click + to add a new profile. Double-click to rename, right-click for more options.

Assignments sidebar

Left panel shows all assignments grouped by modifier layer. Click any item to jump to editing it. The sidebar auto-resets to "All" when you switch profiles.

Modifier bar

Toggle Ctrl Alt Shift Win to select the hotkey layer. The keyboard is dimmed until at least one modifier is active.

Action panel

Opens on the right when you click a key. Shows the action type grid and all form fields for that assignment.

Status bar

Bottom bar shows macros active state, assignment count, and the Listener/Executor engine indicators — both should show green when the engine is running.

Assigning mouse buttons

  1. Click 🖱 Mouse in the view toggle above the keyboard

  2. Optionally select a modifier (Ctrl, Alt, Shift) first to create a layered combination like Ctrl+Back

Left and Right Click

Assigning to bare Left or Right Click replaces their normal function — only do this with a modifier held, e.g. Ctrl+Left Click, or in an app-specific profile where that app handles clicks differently.

Extra buttons (X1–X5)

The X1–X5 slots correspond to the extra side buttons on gaming and productivity mice. If your mouse only has Back/Forward, the X1–X5 slots won't trigger.

Fill-in fields

When an expansion with a fill-in field fires, a small popup asks you to type a value before pasting. The typed value replaces the field in the output. Ideal for email templates where the recipient name changes.

Categories

Organise expansions into named tabs — Email, Legal, CAD, Support — to keep your library manageable.

  1. Click + Category next to the tab bar

  2. Type a name and press Enter

  3. When creating an expansion, use the Category dropdown to assign it

Click a category tab to filter to that group only. The All tab shows everything grouped by category, with Uncategorised items at the top. Right-click a category tab to delete it — expansions move to Uncategorised, not deleted.

Instant trigger

By default expansions fire when you press Space after the trigger word. Instant mode fires the moment the last character is typed — no Space needed.

Enable per expansion using the ⚡ Instant toggle below the trigger field when creating or editing. The expansion list shows a lightning bolt icon on instant-mode items.

Use with care

Instant mode can fire unexpectedly if your trigger word appears inside longer words you type naturally. Use a distinctive prefix like / or . to avoid accidental matches.

Bare key assignments

App-specific profiles unlock a Bare tab in the modifier bar. Bare key hotkeys have no modifier — they fire from a single keypress but only when the linked app is in the foreground.

CAD programs

Use bare keys for single-letter shortcuts that only activate inside the CAD window — they won't interfere with typing in emails or other apps.

Auto-switching pauses while editing

While Trigr is in focus, profile auto-switching is paused. This lets you configure your CAD profile without it switching away mid-edit.

Settings

Access settings via the cog icon ⚙ in the top-right of the title bar.

Compatibility

If text output is not working correctly in a specific application, change the input method:

MethodBest for
Direct KeystrokesCAD software, BricsCAD, engineering tools. Default. Works out of the box.
Clipboard (Shift+Insert)Apps that block Ctrl+V but accept Shift+Insert
Clipboard (Ctrl+V)Standard applications — fastest method
SendInput APIMaximum compatibility — injects at the Windows level

The Pre-execution delay slider (0–500ms) sets how long Trigr waits after a hotkey fires before sending output. Increase if an application needs more time to be ready to receive input.

Quick Search

Press Win+Space (default) from anywhere to open a floating search panel across all your macros and expansions. Change the hotkey in Settings → Quick Search.

Backup & Restore

Use Export Config to save all your assignments, profiles and expansions as a JSON backup file. Use Import Config to restore from backup. Always export before reinstalling or making large structural changes.

Do not store passwords

Avoid storing passwords or credentials as text expansions or macro text. Use a dedicated password manager such as Bitwarden or 1Password. All Trigr data is stored locally on your device only — nothing is ever transmitted to any external server.

Changing the hotkey

Go to Settings → Quick Search and click the hotkey field. Press your desired combination. The default Win+Space conflicts with Windows search on some systems — change it to Ctrl+Space or a function key if needed.

Fire without opening Trigr

Quick Search lets you trigger any macro or expansion even if the Trigr window has never been opened in that session. As long as the tray icon is running, the search panel works.

Troubleshooting

Common issues and how to fix them. If a problem persists, check the status bar — both Listener and Executor chips should be green.

Macros not firing

SymptomLikely causeFix
Nothing happens at allEngine not runningCheck status bar — if Listener/Executor chips are not shown or grey, restart Trigr via the tray icon
Macro fires in some apps but not othersApp intercepts the hotkey before TrigrChange the hotkey to a different combination less likely to conflict
Correct key pressed but wrong macro firesWrong profile activeCheck the profile tab — the active one is highlighted amber. Auto-switching may have changed it.
Hotkey fires but nothing is typedInput method mismatchGo to Settings → Compatibility and switch to Direct Keystrokes or SendInput API

Text output problems

SymptomFix
Text appears with wrong characters or missing lettersTry Clipboard (Ctrl+V) mode — avoids keystroke translation issues
Text pasted in wrong field or wrong windowIncrease the Pre-execution delay in Settings (try 150–300ms)
CAD app ignores typed textUse Direct Keystrokes mode — required by most CAD applications including BricsCAD
Rich text formatting lostThe target app may not support rich text via clipboard. Use the "Paste as plain text" toggle on the expansion.

Text expansions not triggering

SymptomFix
Trigger word typed + Space but nothing happensCheck that Text Expansions are enabled — look for the "TEXT EXPANSIONS" label below the keyboard. It should not be greyed out.
Only fires sometimesMake sure you are pressing Space after the final character with no pause.
Fires inside words unexpectedlyYour trigger word appears as part of longer words. Use a distinctive prefix like / or .
Dynamic date field shows wrong dateVerify system date/time is correct in Windows Settings

Profile auto-switching not working

  1. Confirm the profile has a coloured dot — this indicates it is linked to an app

  2. In profile settings, click Link to App and verify the correct .exe path is set

  3. Make sure Trigr itself is not in focus — auto-switching pauses while you are editing inside Trigr

  4. Some apps launch through launchers or wrappers — link to the actual process .exe, not a shortcut

Antivirus false positives

Some antivirus tools flag keyboard hook software as suspicious. If Trigr stops working after an antivirus scan, add the Trigr install folder to your AV exclusions list. All Trigr code is open source — nothing is sent to any server.

Run as Administrator

A small number of applications — primarily administrative tools and some games — run at elevated privilege and block input from non-elevated processes. If macros don't fire in a specific app, try right-clicking the Trigr tray icon and selecting Restart as Administrator.
